# Wordle Game Clone

A full-stack implementation of the popular word-guessing game Wordle.

## 🎮 Play Now

**[Play the live version here](https://wordle-frontend-g1wu.onrender.com)**

## 📝 Description

This is a modified clone of the popular word game Wordle, where players attempt to guess a 5-letter word in six tries or fewer. After each guess, you get feedback:
- Green tiles indicate correct letters in the correct position
- Yellow tiles indicate correct letters in the wrong position
- Gray tiles indicate letters not in the word

The game also features:
- Streak tracking to keep count of consecutive wins
- Dark/light mode toggle
- Mobile-responsive design
- Dictionary validation for guesses

## 🚀 Local Development

### Prerequisites
- Node.js (v14+ recommended)
- npm or yarn

### Setup

#### Backend
1. Navigate to the backend directory:
   ```
   cd backend
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the backend server:
   ```
   npm start
   ```
   The backend will run on http://localhost:5001

#### Frontend
1. Navigate to the frontend directory:
   ```
   cd frontend
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the frontend development server:
   ```
   npm start
   ```
   The frontend will run on http://localhost:3000

## 🎯 Features

- **Word Validation**: All guesses are checked against a dictionary
- **Game State Management**: Track your progress and statistics
- **Streak Counter**: Monitor your winning streak
- **Responsive Design**: Play on any device
- **Keyboard Support**: Use your physical keyboard or the on-screen keyboard
- **Visual Feedback**: Clear visual feedback for each guess
